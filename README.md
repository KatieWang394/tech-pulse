# ⚡ Tech Pulse

A personal tech intelligence dashboard that tracks trending topics from Hacker News and lets you build a searchable, AI-summarized knowledge base.

## Features
- **Trend Tracker** — Automatically fetches and categorizes trending Hacker News articles using LLM APIs
- **Knowledge Base** — Save any article by URL, get AI-generated summaries and tags
- **Trend Visualization** — See which tech topics are gaining momentum over time

## Tech Stack
Python · Streamlit · SQLite · Claude API · Hacker News API · BeautifulSoup

## Quick Start
```bash
pip install -r requirements.txt
streamlit run app.py
```
